## TODO 5：后端 API

推荐：

# FastAPI

原因：

- AI 生态极强
- Python AI 集成方便
- 开发速度快
- 比 Spring Boot 更适合 AI 时代个人开发

实现：

### 必须功能：

- 用户登录
- JWT
- REST API
- WebSocket
- 文件上传

- ORM

```
SQLAlchemy + Alembic
```
